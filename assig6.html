Step 3 
DELIMITER $$

CREATE PROCEDURE ParameterizedCursor(IN p_min_salary INT)
BEGIN
    DECLARE v_id INT;
    DECLARE v_name VARCHAR(30);
    DECLARE v_salary INT;
    DECLARE done BOOLEAN DEFAULT FALSE;
    
    DECLARE emp_crsr CURSOR FOR
        SELECT emp_id, emp_name, emp_salary
        FROM New_employee
        WHERE emp_salary >= p_min_salary;
    
    DECLARE CONTINUE HANDLER FOR NOT FOUND SET done = TRUE;
    
    OPEN emp_crsr;
    
    read_loop: LOOP
        FETCH emp_crsr INTO v_id, v_name, v_salary;
        
        IF done THEN
            LEAVE read_loop;
        END IF;
        
        INSERT INTO Employee (emp_id, emp_name, emp_salary)
        VALUES (v_id, v_name, v_salary)
        ON DUPLICATE KEY UPDATE
            emp_name = VALUES(emp_name),
            emp_salary = VALUES(emp_salary);
    END LOOP;
    
    CLOSE emp_crsr;
END $$

DELIMITER ;
How it works:

Cursor fetches employees with salary ‚â• threshold, then inserts new records or updates existing ones using ON DUPLICATE KEY UPDATE.

üí° Key Concepts

1. Parameterized Cursor

DECLARE emp_crsr CURSOR FOR
    SELECT ... WHERE emp_salary >= p_min_salary;

Cursor query uses procedure parameter for dynamic filtering.

2. CONTINUE HANDLER

DECLARE CONTINUE HANDLER FOR NOT FOUND SET done = TRUE;

Sets flag when cursor reaches end, allowing graceful loop exit.

3. LOOP with Label

read_loop: LOOP
    FETCH emp_crsr INTO v_id, v_name, v_salary;
    IF done THEN LEAVE read_loop; END IF;
    -- Process record
END LOOP;

Label allows explicit LEAVE statement for loop exit.

4. ON DUPLICATE KEY UPDATE (UPSERT)

INSERT INTO Employee (emp_id, emp_name, emp_salary)
VALUES (v_id, v_name, v_salary)
ON DUPLICATE KEY UPDATE
    emp_name = VALUES(emp_name),
    emp_salary = VALUES(emp_salary);

If PRIMARY KEY exists, UPDATE; otherwise INSERT.

5. Declaration Order

Correct Order: Variables ‚Üí Cursors ‚Üí Handlers

Handlers may reference cursors, cursors may use variables.

6. Cursor Lifecycle

OPEN: Execute query, create result set

FETCH: Get one row, move pointer forward

CLOSE: Release resources (mandatory)
