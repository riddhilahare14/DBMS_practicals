Assignment 5: Student Grading System

📋 Problem Statement

📖 Automated Student Grading System

Create a grading system that accepts student marks, validates them (0-1500), assigns grades, and stores results using both Stored Procedure and Function approaches.

🎯 Grading Criteria (Out of 1500 marks)

Distinction: 990 - 1500 (66%+)

First Class: 900 - 989 (60% - 65.9%)

Higher Second Class: 825 - 899 (55% - 59.9%)

Pass: 0 - 824 (Below 55%)

1️⃣ Database Setup

Step 1 
USE te31145_db;

CREATE TABLE Stud_Marks (
    roll_no INT,
    name VARCHAR(50),
    marks INT
);

CREATE TABLE Result (
    roll_no INT,
    class ENUM('Distinction', 'First Class', 'Higher Second Class', 'Pass')
);
💡 ENUM Data Type:

Restricts values to predefined list, ensures data validation and saves storage space.

2️⃣ Stored Procedure: ProcGrade

Step 2 
DELIMITER $$

CREATE PROCEDURE ProcGrade (
    IN p_rollno INT
)
BEGIN
    DECLARE v_marks INT;
    DECLARE v_grade VARCHAR(30);
    
    DECLARE exit handler FOR NOT FOUND 
    BEGIN
        SELECT 'No matching record found.' AS Message;
    END;
    
    SELECT marks INTO v_marks
    FROM Stud_Marks
    WHERE roll_no = p_rollno;
    
    IF v_marks <= 1500 AND v_marks >= 0 THEN
        IF v_marks >= 990 THEN
            SET v_grade = 'Distinction';
        ELSEIF v_marks >= 900 THEN
            SET v_grade = 'First Class';
        ELSEIF v_marks >= 825 THEN
            SET v_grade = 'Higher Second Class';
        ELSE
            SET v_grade = 'Pass';
        END IF;
        
        INSERT INTO Result (roll_no, class)
        VALUES (p_rollno, v_grade);
        
        SELECT 'Student graded successfully.' AS Message, v_grade AS Grade;
    ELSE
        SELECT 'Marks Out of Range.' AS Message;
    END IF;
END $$

DELIMITER ;

4️⃣ Function: ReturnGrade

Step 4 
DELIMITER $$

CREATE FUNCTION ReturnGrade (
    marks INT
) RETURNS VARCHAR(30)
DETERMINISTIC
BEGIN
    DECLARE grade VARCHAR(30);
    
    IF marks > 1500 OR marks < 0 THEN
        SET grade = 'Invalid marks';
    ELSEIF marks >= 990 THEN
        SET grade = 'Distinction';
    ELSEIF marks >= 900 THEN
        SET grade = 'First class';
    ELSEIF marks >= 825 THEN
        SET grade = 'Higher second class';
    ELSE
        SET grade = 'Pass';
    END IF;
    
    RETURN grade;
END $$

DELIMITER ;
DETERMINISTIC: Required for functions used in SELECT statements. Tells MySQL that same input always produces same output.

Advantage: Functions can be used directly in SELECT to calculate grades on-the-fly without storing them.

⚖️ Procedure vs Function

📦 Stored Procedure
  • ✓ Can modify database (INSERT/UPDATE)
  • ✓ Called with CALL statement
  • ✓ Can have IN, OUT, INOUT parameters
  • ✗ Cannot be used in SELECT
🔧 Function
  • ✓ Can be used in SELECT statements
  • ✓ Must return a single value
  • ✗ Cannot modify database
  • ✗ Only IN parameters allowed

💡 Key Concepts

1. EXIT HANDLER

DECLARE exit handler FOR NOT FOUND 
BEGIN
    SELECT 'No matching record found.' AS Message;
END;

Fires when SELECT INTO finds no matching row. EXIT handler terminates procedure after handling error.

2. SELECT INTO

SELECT marks INTO v_marks
FROM Stud_Marks
WHERE roll_no = p_rollno;

Fetches single value into variable. Must return exactly one row (or zero with handler).

3. IF-ELSEIF Structure

Evaluates conditions from highest to lowest. Order matters - check highest threshold first.
